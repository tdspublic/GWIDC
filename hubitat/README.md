# Hubitat Integration

This directory contains:

- `drivers/GWIDC-Hubitat_driver-v1-20.groovy` — a custom Hubitat device driver
  that communicates with the GWIDC ESP8266 NeoPixel WiFi device.
- `docs/` — Hubitat-specific installation and configuration documentation.
